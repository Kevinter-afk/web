Programming Concepts

A programming language is a tool that allows you to develop software or computer programs. They are used to design and implement programs responsible for defining and managing the behavior of the physical and logical devices of a computer. The above is achieved through the creation and implementation of precision algorithms that are used as a form of human communication with the computer.

They are achieved through:

1) Algorithm

An algorithm is the set of steps logically ordered to carry out an activity or solve a specific problem.

Characteristics:

  • Must have an ending
  • Should give the same result with the same parameters
  • Must be precise

2) Programming Language

It is a set of words, letters, numbers and special symbols with defined syntactic rules that are used to give instructions to the computer.

Examples: Qbasic, Visual Basic, among others.

3) Source Program

It is the set of steps of an algorithm written in a high-level programming language.

4) Object Program

It is the set of steps of an algorithm written in machine language.

5) Compiler

It is an application or program that translates a source program into an object program that can be interpreted and executed by the computer.

6) Pseudocode

It is a set of instructions that are written in words similar to natural language to solve a specific problem or task.

7) Flowchart

It is the graphic representation of an algorithm using conventional symbols that show the actions, processes, and flow of information to solve a given problem.
